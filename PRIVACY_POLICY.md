# Privacy Policy / Политика конфиденциальности

**FunPay Golden Key Manager**

**Last Updated / Последнее обновление:** January 28, 2026

---

## English Version

### What data do we collect?

**We collect NO data.** This extension does not collect, store, or transmit any personal information to external servers.

### What data is stored locally?

The extension stores the following data **locally in your browser only**:

1. **Account names** - The names you assign to your saved accounts
2. **Golden keys** - The authentication tokens you choose to save
3. **Avatar URLs** (optional) - If fetched from FunPay

All this data is stored using Chrome's local storage API and **never leaves your device**.

### What permissions does the extension use?

1. **cookies** - To read and write the `golden_key` cookie on FunPay domains
2. **storage** - To save your account list locally in your browser
3. **activeTab** - To interact with the current FunPay tab

### Does the extension send data anywhere?

**NO.** The extension:
- Does NOT send data to any external servers
- Does NOT track your browsing
- Does NOT collect analytics
- Does NOT share your data with third parties
- Does NOT contain advertisements

### How can I delete my data?

You can delete all stored data at any time by:
1. Removing all saved accounts in the extension
2. Uninstalling the extension from Chrome

### Third-party services

The extension may make requests to FunPay.com to:
- Fetch user information (username, avatar) when you add a new account
- This is done directly between your browser and FunPay

No intermediary servers are used.

### Contact

If you have questions about this privacy policy, please contact us via the Chrome Web Store support page.

## Русская версия

### Какие данные мы собираем?

**Мы НЕ собираем никаких данных.** Это расширение не собирает, не хранит и не передает никакую личную информацию на внешние серверы.

### Какие данные хранятся локально?

Расширение хранит следующие данные **только локально в вашем браузере**:

1. **Имена аккаунтов** - Названия, которые вы даете сохраненным аккаунтам
2. **Golden keys** - Токены авторизации, которые вы решили сохранить
3. **URL аватаров** (опционально) - Если загружены с FunPay

Все эти данные хранятся с использованием локального хранилища Chrome и **никогда не покидают ваше устройство**.

### Какие разрешения использует расширение?

1. **cookies** - Для чтения и записи cookie `golden_key` на доменах FunPay
2. **storage** - Для локального сохранения списка аккаунтов в браузере
3. **activeTab** - Для взаимодействия с текущей вкладкой FunPay

### Отправляет ли расширение данные куда-либо?

**НЕТ.** Расширение:
- НЕ отправляет данные на внешние серверы
- НЕ отслеживает ваши действия в браузере
- НЕ собирает аналитику
- НЕ передает ваши данные третьим лицам
- НЕ содержит рекламы

### Как удалить мои данные?

Вы можете удалить все сохраненные данные в любое время:
1. Удалив все сохраненные аккаунты в расширении
2. Удалив расширение из Chrome

### Сторонние сервисы

Расширение может отправлять запросы на FunPay.com для:
- Получения информации о пользователе (имя, аватар) при добавлении нового аккаунта
- Это происходит напрямую между вашим браузером и FunPay

Промежуточные серверы не используются.

### Контакты

Если у вас есть вопросы по этой политике конфиденциальности, свяжитесь с нами через страницу поддержки в Chrome Web Store.

## Data Security / Безопасность данных

### How is my data protected? / Как защищены мои данные?

- All data is stored locally using Chrome's secure storage API
- No data transmission to external servers
- No third-party access to your stored tokens

### Все данные хранятся локально используя безопасный API Chrome
- Нет передачи данных на внешние серверы
- Нет доступа третьих лиц к вашим токенам

### What happens if I uninstall the extension? / Что происходит при удалении расширения?

All locally stored data (account names, golden keys, avatars) will be automatically deleted when you uninstall the extension.

Все локально хранимые данные (имена аккаунтов, golden keys, аватары) будут автоматически удалены при удалении расширения.
